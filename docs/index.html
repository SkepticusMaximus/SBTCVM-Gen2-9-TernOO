Welcome to the SBTCVM Project Hub

A central reference point for resources related to SBTCVM Gen2-9 (a balanced ternary virtual machine by Thomas Leathers) and the TernOO project (an experimental object-oriented extension to SBTCVM).

Pick an item from the sidebar to load it here. Tutorials and design documents render in this pane. Tools (the Object Inspector and the Opcode Reference) load as interactive widgets. External links to repositories, wikis, and Discord open in a new tab.

SBTCVM is the work of Thomas Leathers. TernOO is an additive extension built collaboratively by Stevo (SkepticusMaximus) and Claude in April 2026.

Both projects are FOSS. The original SBTCVM lives at the link in the sidebar.
